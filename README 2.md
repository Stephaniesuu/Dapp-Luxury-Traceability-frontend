# Dapp—frontend

## quick start

```
# Installation of dependent libraries
yarn install
#or 
npm install

# You can use --force if there are some version conflicts about dependencies 
yarn install --force

# start 
npm run start

# enter url:http://localhost:3000 
# or http://127.0.0.1:3000/ 

```
